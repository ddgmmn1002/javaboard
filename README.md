## 1. 프로젝트 소개

**영화 정보 커뮤니티**

다양한 영화, 드라마 컨텐츠를 소개하고 리뷰를 확인할 수 있는 커뮤니티입니다.

## 2. 기술 스택

- JSP
- Servlet
- Tomcat 9
- jQuery

**Style**
- Bootstrap 5

**Database**
- MariaDB

**API**
- TinyMCE (WYSIWYG editor)
- Youtube (video player)

## 3. 기능

**회원가입과 로그인, 로그아웃**
- 사용자들은 로그인 기능을 통해 비회원 이용자와 회원 이용자로 구분됩니다.
